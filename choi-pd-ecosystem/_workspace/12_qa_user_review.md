## 발견된 이슈 (사용자 관점)

### ISSUE-001: "무료로 시작하기" CTA가 아무 데도 안 감 (Critical)
- **위치**: 홈 히어로 섹션 + 가격표 섹션
- **현상**: "무료로 시작하기" 버튼 클릭 시 /chopd에 머무름. /onboarding으로 이동해야 함
- **사용자 영향**: 첫 방문자가 가입 시도 시 아무 반응 없음 → 즉시 이탈
- **수정 방향**: href="/onboarding"으로 연결

### ISSUE-002: "데모 보기" 버튼도 동작 안 함 (High)
- **위치**: 홈 히어로 섹션
- **현상**: 클릭해도 반응 없음
- **수정 방향**: 데모 영상 섹션으로 스크롤 또는 데모 페이지 연결

### ISSUE-003: 온보딩 "다음" 버튼 disabled 상태 (High)
- **위치**: /onboarding
- **현상**: 직업군을 선택해야 "다음"이 활성화되는데, 직업군 카드 클릭이 작동하는지 불확실
- **사용자 영향**: 가입 퍼널 첫 단계에서 막힘
- **수정 방향**: 직업군 카드 클릭 시 선택 상태 + "다음" 활성화 검증 필요

### ISSUE-004: 교육 과정이 비어있음 — "새로운 교육 과정이 곧 시작됩니다" (Medium)
- **위치**: /chopd/education
- **현상**: 데모/시드 데이터가 없어서 사이트가 텅 비어 보임
- **사용자 영향**: "서비스가 아직 준비 안 됐구나" 인상
- **수정 방향**: 시드 데이터로 샘플 교육 과정 3개 추가

### ISSUE-005: Works 페이지 모든 섹션 빈 상태 (Medium)
- **위치**: /chopd/works
- **현상**: 저서, 갤러리, 언론보도 모두 "준비 중" 또는 "등록되지 않았습니다"
- **사용자 영향**: 포트폴리오 확인 불가
- **수정 방향**: 시드 데이터 또는 placeholder 콘텐츠

### ISSUE-006: 커뮤니티 "첫 번째 소식이 곧 올렸습니다" (Medium)
- **위치**: /chopd/community 공지사항 섹션
- **현상**: 게시물 0개, empty state
- **수정 방향**: 시드 데이터로 공지 2-3개 추가

### ISSUE-007: 문의 폼 제출 피드백 불확실 (Medium)
- **위치**: /chopd/community 문의 폼
- **현상**: 폼 제출 후 성공/실패 피드백이 사용자에게 명확히 전달되는지 확인 필요
- **수정 방향**: 토스트 또는 성공 메시지 표시 확인

### ISSUE-008: 가격표 "Pro 시작하기", "도입 문의하기" 링크 미연결 (Medium)
- **위치**: 홈 가격표 섹션
- **현상**: CTA 버튼 클릭 시 동작 확인 필요
- **수정 방향**: Pro → /onboarding, Biz → /chopd/community 문의 폼

### ISSUE-009: Header 네비게이션에 "미디어" 메뉴 누락 (Low)
- **위치**: 전체 헤더
- **현상**: Education, Works, Community는 있지만 Media 페이지 링크가 보이지 않음
- **수정 방향**: /chopd/media 경로가 있다면 네비에 추가

### ISSUE-010: 모바일에서 가격표 3열이 세로 스택됨 — 가독성 OK 하지만 비교 어려움 (Low)
- **위치**: 모바일 홈
- **현상**: 가격표가 세로로 길어져서 플랜 간 비교가 힘듦
- **수정 방향**: 모바일 전용 요약 비교표 또는 슬라이더
