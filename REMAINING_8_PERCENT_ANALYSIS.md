## 🔍 미완료 8% 상세 내역

### **1. Epic 6 (CMS) - 5% 미완료**

#### 미완성 기능
✅ 완료:
- 게시글 작성 페이지 (마크다운 에디터)
- 게시글 API (GET, POST)
- 실시간 미리보기

❌ 미완료:
1. **게시글 목록 페이지** (`/admin/posts`)
   - 게시글 테이블 뷰
   - 카테고리별 필터링
   - 검색 기능
   - 페이지네이션

2. **게시글 수정 페이지** (`/admin/posts/[id]/edit`)
   - 기존 게시글 불러오기
   - 마크다운 에디터
   - 업데이트 API 연동

3. **게시글 삭제 기능**
   - DELETE API 엔드포인트
   - 삭제 확인 다이얼로그

4. **이미지 업로드**
   - 에디터 내 이미지 업로드
   - GCS 또는 Vercel Blob 통합
   - 드래그 앤 드롭

5. **Rich Text Editor 고도화** (선택사항)
   - Tiptap 또는 Lexical 통합
   - WYSIWYG 편집
   - 표, 코드 블록 지원

**예상 작업 시간**: 4-6시간
**우선순위**: Medium

---

### **2. Epic 10 (알림 시스템) - 40% 미완료**

#### 완료된 부분 (60%)
✅ 데이터베이스 스키마 (`notifications` 테이블)
✅ API 엔드포인트
- `GET /api/notifications?userType=admin`
- `POST /api/notifications`
- `PATCH /api/notifications` (읽음 표시)

#### 미완성 기능 (40%)
1. **알림 센터 UI**
   - `/admin/notifications` 페이지
   - `/pd/notifications` 페이지
   - 알림 목록 테이블
   - 읽음/안읽음 표시
   - 일괄 읽음 처리

2. **헤더 알림 아이콘**
   - 벨 아이콘 (읽지 않은 알림 개수 표시)
   - 드롭다운 메뉴 (최근 5개 알림)
   - "모두 보기" 링크

3. **실시간 알림** (선택사항)
   - Pusher 또는 Socket.io 통합
   - WebSocket 연결
   - 실시간 알림 푸시

4. **이메일 알림** (선택사항)
   - Resend 또는 SendGrid 통합
   - 이메일 템플릿
   - 발송 로직

**예상 작업 시간**: 6-8시간
**우선순위**: Medium

### **3. Epic 8 (멀티 테넌트) - 25% 미완료**

#### 미완성 기능 (25%)

3. **서브도메인별 분석**
